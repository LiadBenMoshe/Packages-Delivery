# Packages-Delivery
Big Data system for sending packages, which includes cloud services, access to databases and analytics with machine learning.


### Technologies
- MongoDB
- Redis
- Docker
- Node.js
- BigMl
- Firebase


#### For install clone this project or download and run ``` npm start```
